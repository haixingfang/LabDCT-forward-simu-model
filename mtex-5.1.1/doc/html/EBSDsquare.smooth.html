smooth

(method of EBSDsquare)

smooth spatial EBSD data

Synatx ebsd = ebsd.gridify ebsd = smooth(ebsd)

Syntax

[ebsd,filter] = smooth(ebsd,'param1',val1,'param2',val2)

Input

ebsd

EBSD

Example
