findByLocation

(method of EBSD)

select EBSD data by spatial coordinates

Syntax

map = findByLocation( ebsd, xy )

Input

ebsd

EBSD
xy

list of [x(:) y(:)] coordinates, respectively [x(:) y(:) z(:)]

Output

ebsd

EBSD subset

Example 

mtexdata fo
plotx2east
plot(ebsd)
p = [10000 5000] %ginput(1)
g = findByLocation(ebsd,p)
