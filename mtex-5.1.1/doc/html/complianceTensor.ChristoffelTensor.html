ChristoffelTensor

(method of complianceTensor)

Christoffel tensor of an elasticity tensor for a given direction

Formula: E_jk = C_ijkl n_j n_l

Syntax

varargout = ChristoffelTensor(S,'param1',val1,'param2',val2)

Input

S

elatic compliance tensor
x

list of vector3d

Output

E

Christoffel tensor
