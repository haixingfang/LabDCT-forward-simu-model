mtexEBSDPath

returns the default path to EBSD data files

Syntax

mtexEBSDPath
getMTEXpref('EBSDPath')
setMTEXpref('EBSDPath',path)
getMTEXpref('EBSDPath') long form of mtexEBSDPath
setMTEXpref('EBSDPath',path) change the path to EBSD files
