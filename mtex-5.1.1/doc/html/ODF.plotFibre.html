plotFibre

(method of ODF)

plot odf along a fibre

Syntax

plotFibre(odf,f);

Input

odf

ODF
f

fibre
param,val Parameters and values that control plotFibre
Parameter Description
'resolution'

resolution of each plot

Example 

odf = SantaFe;
f = fibre.gamma(odf.CS,odf.SS)
plotFibre(SantaFe,f)
 
f = fibre
