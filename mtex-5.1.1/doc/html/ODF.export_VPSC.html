export_VPSC

(method of ODF)

export an ODF to the VPSC format

Syntax 

export_VPSC(odf,'file.txt','points',points)

Input

odf

ODF to be exported
filename

name of the ascii file
param,val Parameters and values that control export_VPSC
Parameter Description
'points'

number of orientations
