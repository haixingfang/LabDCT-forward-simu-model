rotation_special

(method of symmetry)

returns symmetry elements different from rotation about c-axis

Syntax

rot = rotation_special(cs,'param1',val1,'param2',val2)

Input

cs

symmetry

Output

q

symmetry elements other then rotation about the z-axis
rho

position of the mirroring plane
