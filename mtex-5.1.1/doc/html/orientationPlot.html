orientationPlot

ODFSECTIONS

Syntax

oP = orientationPlot(ax,CS1,'param1',val1,'param2',val2)

Example 

cs = crystalSymmetry('mmm')
oS = axisAngleSections(cs,cs)
ori = oS.makeGrid('resolution');
oM = PatalaColorKey(cs,cs)
rgb = oM.orientation2color(ori);
plot(oS,rgb,'surf')
