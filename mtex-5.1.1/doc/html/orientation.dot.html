dot

(method of orientation)

compute minimum dot(o1,o2) modulo symmetry

Syntax 

d = dot(o1,o2)

Input

o1, o2

orientation

Output

d = cos(omega/2) where omega is the smallest rotational angle of inv(o1)*o2

See also

orientation/dot_outer, orientation/angle
