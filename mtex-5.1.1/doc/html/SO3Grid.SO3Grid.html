SO3Grid

(method of SO3Grid)

Syntax

S3G = SO3Grid(nodes,CS,SS) S3G = SO3Grid(points,CS,SS) S3G = SO3Grid(resolution,CS,SS)

Input

points

number of nodes
nodes

quaternion
resolution

double
CS, SS

symmetry groups
param,val Parameters and values that control SO3Grid
Parameter Description
'regular'

construct a regular grid
'equispaced'

construct a equispaced grid
'phi'

use phi
'ZXZ, Bunge'

Bunge (phi1 Phi phi2) convention
'ZYZ, ABG'

Matthies (alpha beta gamma) convention
'maxAngle'

only up to maximum rotational angle
'center'

with respect to this given center
