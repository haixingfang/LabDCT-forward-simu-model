noisepf

(method of PoleFigure)

simulate diffraction counts

noisepf simulates realistic diffraction counts by generating random samples of the Poisson distribution with mean m = alpha * pdf + bg

Syntax 

pdfn = noisepf(pdf,alpha,bg)

Input

pf

PoleFigure
alpha

uniform radiation (double)
bg

background radiation (double)
param,val Parameters and values that control noisepf
Parameter Description
'NONNEGATIV'

force data to be non negative
